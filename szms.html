技术咨询

Data Analysis with R

A comprehensive introduction to R for data wrangling, data mining, visualization, and reporting. Learn skills from basic programming elements and syntax to advanced packages, such as tidyr, dplyr, ggplot2, knitr, …

R Shiny

Customized R Shiny training and end-to-end application solutions, from basics R Shiny elements, to customizing Shiny App appearance, modularizing code and hosting apps.

Data Analysis with Python

An introduction to data analysis with Python. Covers the most popular data analytics and visualization modules, including numpy, scipy, pandas, matplotlib, and Beautifulsoup.

Machine Learning with Python

Machine Learning skills with scikit-learn with focus on data cleaning, feature extraction, natural language processing, model selection, SVM, PCA, tree models, clustering and more

Hadoop Ecosystem

Learn the concepts of distributed computing and Hadoop Distributed File System. Implement MapReduce with Python Streaming. Learn key components of Hadoop, including HDFS, Hive, Pig and AWS.

Spark

Introduction to Spark, Spark MLlib and Spark SQL.
